Metric details

MRR_url_level

Mean Reciprocal Rank at URL level. For each question, rank position r of first correct URL. MRR = (1/Q) * sum_{i=1..Q} (1/r_i).

Precision@10

Fraction of questions where ground-truth URL appears in top-10 retrieved URLs: Precision@10 = (1/Q) * sum_{i} 1[ground in top10].

SemanticAnswerMetric

If BERTScore is available, mean BERTScore F1 between generated answer and reference answer; otherwise token-level F1 (precision/recall harmonic mean).
